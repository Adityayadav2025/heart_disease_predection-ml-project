Heart Disease Prediction System using Machine Learning
Overview

The Heart Disease Prediction System is a Machine Learning-powered web application that predicts the likelihood of heart disease based on a patient's clinical information. The application utilizes multiple machine learning models and an ensemble prediction approach to provide accurate risk assessments.

The system is built using Flask for the backend, SQLite for data storage, and Machine Learning models including Random Forest, Logistic Regression, and Multi-Layer Perceptron (MLP).

Features
🔹 Machine Learning-Based Prediction
Predicts heart disease risk using patient health parameters.
Uses multiple ML models for improved reliability.
Ensemble prediction combines outputs from all models.
🔹 Interactive Web Interface
User-friendly interface for entering patient details.
Instant prediction results through REST APIs.
🔹 Risk Assessment
Calculates heart disease probability.
Classifies patients into:
Safe
High Risk
🔹 Data Storage
Stores prediction records in SQLite database.
Maintains prediction history with timestamps.
🔹 API Support
RESTful endpoints for prediction and record retrieval.
JSON-based communication.
Machine Learning Models Used
1. Random Forest Classifier
Ensemble learning algorithm.
Handles complex relationships effectively.
Reduces overfitting through multiple decision trees.
2. Logistic Regression
Statistical classification model.
Efficient and interpretable.
Produces probability scores.
3. Multi-Layer Perceptron (MLP)
Neural Network-based classifier.
Learns non-linear relationships in medical data.
Improves predictive capability.
Ensemble Model

The final prediction is calculated by averaging the probabilities generated by:

Random Forest
Logistic Regression
MLP Classifier

This approach improves robustness and prediction accuracy.

Input Parameters

The system accepts the following medical attributes:

Feature	Description
Age	Patient age
Sex	Male/Female
Chest Pain Type	Type of chest pain
Resting Blood Pressure	Blood pressure at rest
Cholesterol	Serum cholesterol level
Fasting Blood Sugar	Blood sugar level
Resting ECG	Electrocardiographic results
Maximum Heart Rate	Maximum heart rate achieved
Exercise Induced Angina	Chest pain during exercise
Oldpeak	ST depression induced by exercise
Slope	Slope of peak exercise ST segment
CA	Number of major vessels colored
Thal	Thalassemia condition

API Endpoints
Home Page
GET /

Loads the web application interface.

Get Prediction Records
GET /api/records

Returns the latest prediction records.

Response:

[
  {
    "id": 1,
    "age": 45,
    "risk_score": 72.5,
    "status": "High Risk"
  }
]
Predict Heart Disease
POST /predict

Request:

{
  "age": 45,
  "sex": "Male",
  "trestbps": 130,
  "chol": 220
}

Response:

{
  "random_forest": {
    "probability": 0.82,
    "prediction": 1
  },
  "logistic": {
    "probability": 0.76,
    "prediction": 1
  },
  "mlp": {
    "probability": 0.79,
    "prediction": 1
  },
  "ensemble_prob": 0.79
}
